
### Моделирование события с вероятностью ( p )

```
r = random.random()
if r < p:
    return "ДА"
else:
    return "НЕТ"
```

Генератор `random.random()` возвращает число из интервала ([0,1]), равномерно распределённое.

Если вероятность события равна (p), то:

Событие происходит, если случ. число меньше (p);
иначе не происходит.

## Выбор события из всех
```
def choose_event(probs):
    r = random.random()
    s = 0
    for i, p in enumerate(probs):
        s += p
        if r < s:
            return i
```

Пусть есть вероятности, разбиваем 1 на их кол-во и получаются интервалы с заданной вероятностью,
если сч попало в интервал это событие происходит
---
### 1. Корректность распределения

реализуется через разбиение интервала

### 2. Равномерное распределение
```
r = random.random()
```

* все значения равновероятны

### 3. Полная группа событий

сумма вероятностей должна быть равна 1
иначе распределение некорректно

### 4. Независимость испытаний
каждый вызов генератора независим
предыдущие значения не влияют
